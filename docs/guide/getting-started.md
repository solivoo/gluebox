# Introducción

gluBox es una librería de componentes React pensada para dashboards y apps internas.

## Qué incluye hoy

- **Sidebar**: navegación con submenús, permisos RBAC, temas, acordeón, ítems top/bottom.

## Requisitos

- React 18+
- TypeScript recomendado

## Demo local

```bash
git clone https://github.com/solivoo/gluebox.git
cd gluebox
pnpm install
pnpm dev          # App demo
pnpm storybook    # Playground de componentes
pnpm docs:dev     # Sitio de documentación
```
