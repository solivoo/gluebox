# gluBox

Librería de componentes React para aplicaciones empresariales.

## Sidebar

Navegación lateral con:

- Menú dinámico desde API (`MenuConfig`)
- RBAC — filtrado por permisos del usuario
- Hasta **3 niveles**: módulo → opción → acción
- Temas `dark` / `light` / personalizados
- Modo colapsado (rail de iconos)
- Integración con router vía `activePath` y `onNavigate`

## Instalación

```bash
pnpm add glubox
```

```tsx
import { Sidebar } from 'glubox';
import 'glubox/style.css';
```
