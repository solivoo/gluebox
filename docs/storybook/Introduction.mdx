# gluBox

Librería de componentes React para aplicaciones empresariales.

## Características

- **Sidebar** con menú dinámico, submenús y RBAC
- Temas personalizables (`dark` / `light` / custom)
- Iconos inyectables vía `renderIcon`
- Marca personalizable vía componente `brand`
- Ítems `top` / `bottom` en el menú

## Instalación

```bash
pnpm add glubox
```

```tsx
import { Sidebar } from 'glubox';
import 'glubox/style.css';
```
